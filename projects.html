My Projects

What-To-Do

It is my 1st ever programming project. It was my first semester, Programming Fundamentals course project. I made a to-do list app in C++. I implemented the concepts of variables, conditionals, loops and file handling studied in the class in this project. The user can add, remove, update and delete his tasks. Also, he can reschedule his tasks. He can mark tasks done and undone. Moreover, he can see his over-due tasks along with due today, due tomorrow and due later tasks.

Tech Stack: C++

Banking Management System

It was my second semester, Object Oriented Programming course project. I implemented the concepts of OOP in C++ such as inheritance, polymorphism, association, aggregation, and composition. I implemented multiple classes and associated them via each other. I created a menu based project that had separate interface for bankers, customers and company employees. Customers can deposit and withdraw cash. The banker can approve new customers and assign them a card number. The company employees can share a same account.

Tech Stack: C++

Flappy Bird

This is my Computer Organization and Assembly Language project. I created an animation based game using x86 assembly language. I created all the event loops myself. Used global variables for configurations and kept the code as modular as possible. I implemented flying as well as crash logic.

Tech Stack: x86 assembly

Grubify

This is my Database Lab project. It was a team project. My team members implemented backend and frontend logic in REACT.js and Node.js, while I dealt with databases. I worked in MsSQL. I created the schema and wrote stored procedures, triggers, views and enforced business rules via transactions. At the end, I normalized the schema to BCNF.

Tech Stack: REACT.js, Node.js, MsSQL

TripMate

It was my Software Design and Analysis course project. It was also a team project. It was an expense tracker and divider app. The main focus was to design the system first, then implement it. We implemented our project using REACT.js for frontend and Flask for backend, with SQLAlchemy for database.

Tech Stack: REACT.js, Flask, SQLAlchemy

summarizeThis

This was a chat summarization project, using concepts of Natural Language Processing in python. We first used a bi-LSTM model, which was very bad at this task. Then, we used BERT to perform this task and we got pretty good accuracy with fine summaries.

Tech Stack: Python
